# drdx-stuff
Compile [source](https://github.com/pixeljam/DinoRunDX) for macOS, Windows, iOS (in progress), Android (planned).

## dependencies
- macOS 13
- WINE (Windows)
- Adobe AIR 32 compiler
- JDK 8 (macOS), 19 (Windows)
- FRESteamworks, Steamworks SDK (desktop)
